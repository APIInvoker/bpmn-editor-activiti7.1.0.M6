# BPMN Editor

This project is a BPMN editor with a properties panel.

## Requirements

- Node.js >= 20.18.1
- npm >= 10.8.2

## Installation

To install the dependencies, run:

```bash
npm install
```

## Development

To start the development server, run:

```bash
npm start
```

## Build

To build the project for production, run:

```bash
npm run build
```
